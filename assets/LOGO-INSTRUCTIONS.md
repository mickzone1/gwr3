# Logo Instructions

## Required Logo File

Please save the EdUHK logo as **`eduhk-logo.png`** in this `assets` folder.

### Logo Specifications:
- **Format**: PNG (transparent background recommended)
- **Size**: 300×100 pixels (or similar aspect ratio)
- **Location**: `./assets/eduhk-logo.png`

### Current Logo:
The logo you provided shows:
- Green stacked books design with orange circle
- Text: "香港教育大學 The Education University of Hong Kong"
- Orange horizontal line separator

### How to Add the Logo:

1. **Save the logo image** you provided as `eduhk-logo.png`
2. **Place it in the `assets` folder** (same directory as this README)
3. **The app will automatically display it** in the top right of the header

### Alternative - Use Logo URL:

If you prefer to use a hosted logo, update `index.html`:

```html
<img src="https://your-domain.com/eduhk-logo.png" alt="EdUHK" class="header-logo">
```

---

**Note**: The logo is currently displayed in the top right corner of the header and will automatically resize for mobile devices.
