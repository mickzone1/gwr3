# Audio Assets Directory

Place your karaoke background music files in this directory.

## Required File:
- `karaoke-track.mp3` - Your main background audio track

## Supported Formats:
- MP3 (recommended)
- WAV
- OGG
- M4A

## Tips for Audio Files:
1. Use high-quality audio (192kbps or higher for MP3)
2. Ensure the audio length matches your lyrics timestamps
3. Test the audio file in multiple browsers before deployment
4. Consider using royalty-free or properly licensed music

## Where to Get Karaoke Tracks:
- Karaoke Version (karaoke-version.com)
- Suno AI (for AI-generated tracks)
- YouTube Audio Library (free, royalty-free)
- Epidemic Sound (subscription)
- Create your own using DAW software
